# Codex Prompt Library

Requirements:
- App router
- Clean folder structure
- Environment variables
- Placeholder pages

- users
- companies
- products
- categories
- rfqs
- subscriptions
- payments

## RFQ Module

- RFQ form
- Database insert
- Email notification
- Admin RFQ list
