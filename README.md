# Turkey Manufacturers Platform

A B2B sourcing marketplace connecting Turkish manufacturers with global buyers.

## Core Features
- Supplier directory
- Product catalog
- RFQ (Request for Quotation) system
- Supplier subscription plans
- Admin management panel

## Tech Stack
- Next.js
- TypeScript
- PostgreSQL
- Supabase Auth
- Tailwind CSS
- Vercel Deployment

## Documentation
See the `/docs` folder for full technical documentation.

## Project Goals
Build a scalable B2B sourcing platform that helps international buyers discover
Turkish suppliers and request quotations directly.
